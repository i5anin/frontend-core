# Определение

CSS-переменные (Custom Properties) — это именованные значения, которые задаются в стилях и доступны через функцию `var()`.

Переменные работают на уровне **CSSOM**, а не препроцессора — меняются в рантайме.

---

## Объявление переменной

Переменная начинается с `--`.

```css
:root {
  --primary-color: #1a1a1a;
  --gap: 16px;
}
```

`:root` используется для глобальной области видимости.

---

## Использование переменной

```css
button {
  color: var(--primary-color);
  margin-left: var(--gap);
}
```

---

## Значения по умолчанию

Если переменная не найдена:

```css
color: var(--missing, #000);
```

---

## Область видимости

Переменные наследуются по дереву DOM.

```css
.container {
  --bg: #fff;
}

.container .item {
  background: var(--bg);
}
```

Определение внутри элемента переопределяет `:root`.

---

## Динамическое изменение

CSS-переменные изменяются через JS прямым назначением стиля — **без пересборки стилей всего документа**.

```js
document.documentElement.style.setProperty('--primary-color', '#ff0000');
```

---

## Отличия от препроцессоров (SASS/LESS)

- SASS-переменные работают **до компиляции**, CSS-переменные — **в браузере**.
    
- CSS-переменные поддерживают наследование и контекст.
    
- Позволяют создавать динамические темы: тёмная/светлая.
    

---

## Частые кейсы использования

- Темизация (цвета, фон, размеры).
    
- Адаптивность (`clamp()`, `calc()`).
    
- Управление spacing (gap, размеры).
    
- Динамика через JS (например, реагирование на скролл).
    

---

## Важные ограничения

- Имя переменной всегда строка.
    
- Значение не проверяется — браузер подставляет как есть.
    
- Нельзя использовать внутри некоторых структурных свойств, требующих предварительного вычисления (например, селекторы).
