Flexbox — однорядная модель раскладки, где элементы распределяются в одном направлении: по главной оси (main axis). Контейнер управляет выравниванием, порядком и поведением элементов.

**Включение flex-контейнера**

```css
.container {
  display: flex;
}
```

**Главные свойства контейнера**  
`flex-direction` — направление:  
`row` (по горизонтали), `column` (по вертикали).

`justify-content` — выравнивание по главной оси:  
`flex-start`, `center`, `flex-end`, `space-between`, `space-around`, `space-evenly`.

`align-items` — выравнивание по поперечной оси:  
`flex-start`, `center`, `flex-end`, `stretch`, `baseline`.

`flex-wrap` — перенос элементов:  
`nowrap`, `wrap`, `wrap-reverse`.

`gap` — расстояние между элементами (предпочтительно вместо margin-хаков).

**Свойства элементов**  
`flex-grow` — коэффициент растягивания.  
`flex-shrink` — коэффициент сжатия.  
`flex-basis` — базовый размер элемента.  
Комбинированная краткая запись:

```css
flex: 1 0 200px;
```

`order` — изменение визуального порядка без изменения DOM.

**Особенности поведения**

- Размеры элементов вычисляются динамически, в зависимости от свободного пространства.
    
- Элементы автоматически подстраиваются под доступное место.
    
- В отличие от Grid, Flexbox решает раскладку **в одном направлении**, а не в двух.
    
- Отлично подходит для кнопок, панелей, навигации, списков, карточек, центровки элементов.
    

**Типичные паттерны**  
Центрирование по двум осям:

```css
display: flex;
justify-content: center;
align-items: center;
```

Гибкая строка элементов:

```css
display: flex;
flex-wrap: wrap;
gap: 16px;
```

Flexbox — ключевой инструмент для построения современных интерфейсов, позволяющий быстро управлять геометрией и выравниванием элементов без сложных хаков.
